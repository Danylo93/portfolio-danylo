# Danylo Oliveira Portfolio

Portfólio em React + Vite com foco em DevOps, infraestrutura e automação.

## Desenvolvimento local

```sh
npm install
npm run dev
```

## Build de produção

```sh
npm run build
```

## Deploy na Vercel

O projeto já está preparado para Vercel com:

- build command: `npm run build`
- output directory: `dist`
- rewrite para SPA em `vercel.json`

Se quiser publicar via CLI:

```sh
npm i -g vercel
vercel
vercel --prod
```

## Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
